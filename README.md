# Geared Energy Website

A professional website for Geared Energy - boiler solutions, servicing, and spare parts supplier in Kenya.

## Features

- Modern React + Vite setup
- Tailwind CSS for styling
- Responsive design
- Four main pages: Home, Services, About, Contact
- Professional email addresses support

## Pages

1. **Home**: Overview and key services
2. **Services**: Products and services offered
3. **About**: Company information and values
4. **Contact**: Contact form and information
